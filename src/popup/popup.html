ETS Power Tools

Enhance your ETS platform experience with advanced data organization and visualization features.

Features:

  • Data Grouping: Groups table rows by date, parsing and processing the data within each row. Calculates totals for specific metrics such as STD (Standard Time) and OVR (Overtime).
  • Status Marking: Highlights rows with total STD hours of 8 or more, indicating completed workdays. Marks rows with incomplete hours, showing the total logged hours out of the required 8.
  • Visual Enhancements: Adds custom classes to rows to visually differentiate date headers and status indicators. Displays total hours directly within the table for quick reference.

Usage: Once installed, ETS Power Tools will automatically activate when you visit a supported ETS platform. The extension runs a content script that processes the table data, groups rows by date, calculates totals, and marks rows based on completion status. Users can easily identify which days have incomplete hours and which have been fully logged.

Getting Started:

  1. Install the ETS Power Tools extension from the Chrome Web Store.
  2. Navigate to your ETS platform and let the extension automatically process the table data.
  3. Review the enhanced table to quickly understand your work logs and statuses.
